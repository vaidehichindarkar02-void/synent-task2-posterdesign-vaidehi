# Task 2: Poster Design

## Overview
This repository contains my submission for Task 2 of the Synent Graphic Design Internship. The task was to design a creative and informative poster that is visually attractive and easy to understand.

## Objective
The aim of this task was to create a poster with a clean layout, attractive colors, and proper typography while keeping the design professional and engaging.

## My Approach
Before starting the design, I decided on the theme and collected suitable visual elements. I selected a color palette that matched the overall concept and arranged the content to maintain a good visual flow. I also paid attention to spacing, font selection, and alignment so that the poster would look balanced and easy to read.

## What I Learned
Working on this task helped me improve my understanding of layout design, color combinations, typography, and how to communicate information effectively through a poster.
